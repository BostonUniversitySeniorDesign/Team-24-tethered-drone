# Hardware Requirements

## System Requirements

As this project was solely code based, the only hardware that is required is a local machine that can run the necessary programs and languages.

Python and Node.js do not have any specific system requirements.

X-Plane has the following system requirements:

**Minimum Requirements**
- Requires a 64-bit processor and operating system
- OS: Windows 7, 8.1, or 10, 64-bit
- Processor: Intel Core i3, i5, or i7 CPU with 2 or more cores, or AMD equivalent
- Memory: 8 GB RAM
- Graphics: DirectX 11-capable video card from NVIDIA or AMD w/512 MB VRAM
- DirectX: Version 11
- Storage: 20 GB available space
- Sound Card: Default
- Admin permissions to change environment variables (may require a non-work computer)

**Recommended Requirements**
- Requires a 64-bit processor and operating system
- OS: Windows 7, 8.1, or 10, 64-bit
- Processor: Intel Core i5 6600K at 3.5 ghz or faster
- Memory: 16 GB RAM
- Graphics: DirectX 12-capable video card from NVIDIA or AMD w/4 GB VRAM
- DirectX: Version 12
- Network: Broadband Internet connection
- Storage: 65 GB available space
- Sound Card: Default
- Admin permissions to change environment variables (may require a non-work computer)

Please note that the disk requirements are based on how many world scenery areas are downloaded. Minimum with one scenery area is 20 GB .
